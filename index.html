Ecommerce Store

REACT | MATERIAL UI | NODE | EXPRESS | COMMERCE.JS | STRIPE | JWT | POSTGRESQL

An Ecommerce site built using commerce.js API, a headless ecommerce solution. For this project I wanted to put many of my skills to use to deliver an easy and seamless shopping experience. I really enjoyed learning about JWT and how web tokens and authentication/authorization integrates into the backend. The most challenging part of this project was combining different techonolgies together such as commerce.js and stripe and figuring out how each API works. (Note: App hosted on free Heroku server. Please allow 10-15s to spin up.)

My Dictionary

REACT | MATERIAL UI | NODE | EXPRESS | WORDNIK API

I learned a lot about working with an API in this project and enjoyed the process of learning some back-end development with Node. (Note: App hosted on free Heroku server.
